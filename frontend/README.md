# ShopHub Frontend

A modern, responsive e-commerce frontend built with React, Vite, Tailwind CSS, and shadcn/ui.

## Features

- 📱 Responsive design for all devices
- 🛒 Full shopping cart functionality
- 👤 User authentication (Login/Register)
- 📦 Product listings and details
- 💸 Checkout process (integrated with Stripe)
- 📊 Seller dashboard

## Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Icons**: Lucide React
- **State Management**: React Query
- **Animations**: Framer Motion
- **HTTP Client**: Axios

## Getting Started

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Access the application at `http://localhost:8080`.

## Backend Integration

This frontend is designed to work with the Django REST Framework backend located in the `backend/` directory. Ensure the backend is running at `http://localhost:8000`.
